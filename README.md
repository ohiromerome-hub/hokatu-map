# あいち保活マップ — 愛知県の保育園・こども園・幼稚園を空き状況つき地図で比較

🌐 **公開URL: https://ohiromerome-hub.github.io/hokatu-map/**

引越し・転園・保活中の子育て世帯向けに、愛知県内の保育・幼児教育施設を
**地図上で種類別に比較**でき、対応市区は**最新の空き状況（募集人数）**まで見られる静的Webアプリ。
サーバー不要・ビルド不要・無料。公的オープンデータ準拠。

## 機能

| 機能 | 内容 |
|------|------|
| 施設マップ | 保育園1,293・小規模/認可外1,159・認定こども園349・幼稚園365 ＝ **計3,166施設**（色分け・クラスタリング表示） |
| **空き状況** | **名古屋16区＋豊田・岡崎・豊橋・一宮・春日井ほか計30市区**で年齢別の空き状況を表示。「空きあり施設のみ」フィルタつき。公表時点を明記し、取込停止時は自動で非表示（古い空き情報を出さない設計） |
| 市区町村比較 | 69市区町村（名古屋市は区単位）×種別の施設数テーブル。ソート・タップで地図移動 |
| 電車レイヤー | 愛知県周辺の鉄道路線（事業者別色分け）＋駅854。駅名ツールチップ |
| バスレイヤー | バス停9,121＋バスルート13,014区間（トグルON時に遅延読込） |
| 経路検索 | 引越し候補ピン→近い施設トップ10（直線距離・徒歩分数）→各施設へ**Googleマップ乗換案内**を1タップ起動。施設ごとに最寄駅・徒歩分数を自動計算 |
| 検索 | 施設名・市区町村・駅名のインクリメンタル検索＋住所ジオコーディング（Nominatim） |
| 入園案内リンク | 施設ポップアップから市町村公式の入園案内ページへ直リンク・ここdeサーチへの導線 |

> 施設情報・空き状況は公的オープンデータに基づく参考情報です。入園可否・最新の空き状況は必ず各市町村・各施設の公式情報でご確認ください。

## 起動

ローカル: このフォルダで `python3 -m http.server 8080` → http://localhost:8080
（fetchを使うため file:// 直開きでは動かない）

## データの更新（tools/）

```bash
bash tools/download_mlit.sh        # 鉄道N02・バス停P11・バスルートN07
bash tools/download_facilities.sh # 学校P29（幼稚園・こども園）・福祉施設P14（保育所）
python3 tools/convert.py           # → out/*.geojson を data/ にコピー
```

## データ出典・ライセンス

すべて出典明記のうえ**商用利用可**：

| データ | 出典 | 更新 |
|--------|------|------|
| **保育施設（認可・認可外・こども園・給付型幼稚園）** | **ここdeサーチ「全国施設CSV」（こども家庭庁）** — 検索画面フッターの一括DL（`ANN010104E06.do`）。営業中のみ採用・ベビーシッター等の居宅訪問型は除外 | 日次更新（取得日ベース） |
| 幼稚園（私学助成園の補完） | 国土数値情報 学校データ P29（国土交通省） | 2023 |
| 市公式パッチ | 各市の公式施設一覧（tools/patches/） | 随時 |
| ジオコーディング | 国土地理院 住所検索API（キャッシュ: tools/../geocache） | - |
| 鉄道路線・駅 | 国土数値情報 鉄道データ N02（国土交通省） | 2023 |
| バス停留所／バスルート | 国土数値情報 P11／N07（国土交通省） | 2022 |
| 背景地図 | © OpenStreetMap contributors（ODbL） | - |
| 住所検索（アプリ内） | Nominatim（OSM。利用規約：1req/s・大量利用は自前ホスト） | - |

※ 旧版で使用していたOSM施設データ・国土数値情報P14（保育所）は、市町村カバレッジの欠落（51市町村）が判明したため**ここdeサーチ公式CSVに全面置換**（2026-07-03）

- 国土数値情報は[利用規約](https://nlftp.mlit.go.jp/ksj/other/agreement.html)に基づき出典表示のうえ利用（アプリ内フッターに表示済み）
- **空き状況・入園可否は必ず市町村公式で確認**する旨をアプリ内に明記済み（データは静的なため）

## 商用化ロードマップ（メモ）

1. **今のまま公開**：GitHub Pages（¥0）。独自ドメイン＋AdSense/アフィリエイト（引越し・不動産・学資保険系ASP）
2. **空き状況の定期更新**：主要市の空き状況CSV/PDFを月次でスクレイプ→facilities.geojsonに空き列を追加（GitHub Actionsで自動化。ラッコKW収集と同パターン）
3. **本格経路検索**：GTFSデータ＋OpenTripPlannerを立てると乗換案内を内蔵可（サーバー費用が発生するため、当面はGoogleマップ連携で十分）
4. **他県展開**：convert.pyの県コード（23）を変えるだけで47都道府県分を量産可能
5. Nominatimの商用利用が増えたら自前ジオコーダ or 有料API（HeartRails等）へ切替
